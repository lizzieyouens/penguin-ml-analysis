# Penguin Species Classification using Machine Learning

This project investigates whether morphological measurements
of penguins can be used to predict species classification
using supervised machine learning methods.

## Research Question
Can bill length, bill depth, flipper length, and body mass
be used to predict penguin species?

## Dataset
Palmer Penguins dataset (CSV stored locally in the `data/` folder).

## Methods
- Logistic Regression
- Decision Tree Classification

## Files
- `analysis.py` : Main Python analysis code
- `data/penguins.csv` : Dataset
- `README.md` : This file
